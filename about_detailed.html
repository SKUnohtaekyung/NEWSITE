Skills & Tools.

프로젝트를 통해 갈고닦은 기술 스택과 디자인 툴에 대한 저의 경험과 생각을 정리했습니다.

REACT

컴포넌트 기반 아키텍처를 활용하여 유지보수가 용이하고 확장성 높은 웹 애플리케이션을 구축합니다. 상태 관리 라이브러리(Redux, Zustand)와 연동하여 복잡한 상태 로직을 효율적으로 관리한 경험이 있습니다.

JAVASCRIPT

ES6+ 최신 문법에 익숙하며, 비동기 처리(Async/Await, Promise)를 능숙하게 다룹니다. Vanilla JS를 사용하여 인터랙티브한 UI를 구현하고 DOM을 효과적으로 제어할 수 있습니다.

TS

TYPESCRIPT

정적 타이핑을 통해 코드의 안정성을 높이고, 대규모 프로젝트에서의 협업 효율성을 극대화합니다. 인터페이스와 제네릭을 활용하여 재사용 가능하고 예측 가능한 코드를 작성하는 것을 선호합니다.

FIGMA

아이디어 구상부터 와이어프레이밍, 프로토타이핑, 최종 UI 디자인까지 전 과정을 Figma로 진행합니다. 디자인 시스템을 구축하고 컴포넌트 기반으로 작업하여 개발팀과의 원활한 협업을 이끌어낸 경험이 있습니다.
